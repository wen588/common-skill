🧠 One-liner: This is like a doorbell that detects whether your front door is open or closed — and shows a sign so everyone can see.

🏠 The Big Picture: Imagine a small shop with a door sensor. When someone opens the door, the sensor flips a sign from "Open" to "Closed" (or the other way around). The shopkeeper doesn't need to check manually — the sensor just watches the door and updates the sign automatically. This React component does exactly that for your internet connection: it hangs a sensor on your browser window, and whenever your internet goes on or offline, it flips a little "Online / Offline" display.

👣 Walkthrough:

1. `const [isOnline, setIsOnline] = useState(true)`
   → In real life: You set up a sign that starts with "Open" by default, assuming the door is open until told otherwise.
   → In code: This creates a variable called `isOnline` that starts as `true` (assuming you're online), with a配套 switch (`setIsOnline`) to flip it.

2. `useEffect(() => { ... }, [])`
   → In real life: You mount a door sensor once when the shop opens, and it keeps watching forever. You also make sure to remove it when the shop closes, so it doesn't keep beeping after hours.
   → In code: This runs the setup code once when the component first appears, and the `return () => {...}` is the cleanup — it removes the sensor when the component goes away, preventing ghost listeners.

3. `window.addEventListener('online', ...)` and `window.addEventListener('offline', ...)`
   → In real life: You wire the sensor to two signals — "door opened" triggers one reaction, "door closed" triggers the opposite.
   → In code: You tell the browser: "If the internet comes back, call `handleOnline`. If it disconnects, call `handleOffline`." Each handler just flips the sign (`setIsOnline(true/false)`).

4. `{isOnline ? '✅ Online' : '❌ Offline'}`
   → In real life: The sign shows either "✅ Open" or "❌ Closed" — never both, never blank.
   → In code: This is a conditional display — if `isOnline` is true, show the checkmark + "Online"; otherwise show the cross + "Offline".

📝 Summary: This code is a automatic connection indicator — it hangs a sensor on the browser, listens for internet changes, and keeps a sign updated so you always know your online status at a glance.
